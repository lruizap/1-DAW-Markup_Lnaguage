 <!-- fichero HTML -->
 <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Tangerine">

/* fichero CSS */ 
.googlefonts {

@font-face

En caso de disponer del fichero con la fuente se incluye dicha fuente con la directiva @font-face. El nombre que se le de en la directiva es el que se debe usar luego en la propiedad font-family:


@font-face { 
  font-family: nuevaFuente; 
  src: url("mona_shark/Mona Shark.otf");
} 
.comic { 
  font-family: nuevaFuente; 
}
