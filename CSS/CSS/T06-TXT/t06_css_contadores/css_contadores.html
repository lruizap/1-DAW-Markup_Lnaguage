Contadores CSS

Nota: se usan varias cabeceras h1 para ejemplos de uso..

El uso de contadores se basa en el pesudo elemento ::before (o ::after), la propiedad content para insertar el contador y el uso de nuevas propiedades:

  • counter-reset - Creates or resets a counter
  • counter-increment - Increments a counter value
  • counter() - Adds the value of a counter to an element

Si va a numerar los apartados de nivel 2, debe iniciar el contador en una regla CSS asociada al elemento padre, e incrementarlo y mostrarlo en una propiedad del elemento a numerar usando la pseudoclase ::before

Por ejemplo, si quiere numerar los apartados de nivel 2 de una sección de clase unnivel

 
section.unnivel { 
    counter-reset: nh2;
}
section.unnivel h2::before {
    counter-increment: nh2;
    content: "Apartado " counter(nh2) ": ";
}

También se puede crear contadores anidados, usarlo para numerar listas, numerar párrafos, listas de definiciones, etc. Tiene varios ejemplos abajo, y puede obtener más información en:

  • Contadores en W3Schools
  • Ejemplos detallados

Ejemplo de numeración de listas ordenadas

Los contadores son auto-anidados: si se resetea (define) un contador en un elemento descendiente se crea una nueva instancia de ese contador (un contador distinto pero con el mismo identificador).

Esto es útil por ejemplo con las listas anidadas donde es posible que no conozcamos los niveles de anidación, o estos cambien, o sean tantos que hagan el código CSS farragoso.

En este contexto es útil la función counters(): crea un string con todos los contadores que existen en ese ámbito o scope con un identificador dado ,separados por el string que se indica en la función: 

 
.listas ol {
	counter-reset: nli;
	list-style-type: none;
}
.listas li::before {
	counter-increment: nli;
	content: counters(nli,".") " - ";
}
