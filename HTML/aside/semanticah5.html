Introducción

En esta página se usan y explican algunas de los elementos introducidos en HTML para dar semántica adicional a la estructura de una página web. Entre ellos tenemos

  • header
  • nav
  • aside
  • footer
  • section
  • article

Etiqueta header (esto en un h1 en header)

La etiqueta header se usa para marcar un grupo de elementos de introducción dentro de un documento HTML, pero también puede aparecer en una section, article, footer, nav, aside.

Su uso más frecuente es sustituir al habitual <div id="header">, donde se muestra el título de la página, una introducción, el logo de la empresa,..

Etiqueta nav

La etiqueta nav se usa para marcar una sección del documento cuya función es la navegación por la página o el sitio web.

No es obligatorioa que sea una lista de enlaces (aunque es lo más frecuente), ni toda lista de enlaces tiene porqué ser un elemento nav.

Incluso es posible incluir varios elementos nav en la página, por ejmplo uno para navegaciñondentro de la propia página y otro para navegar dentro del sitio web.

Etiqueta article: ejemplo uso artículo dentro sección

Nota: lo normal es que si hay una cabecera que sólo contiene una cabecera de nivel no se usa la etiqueta header, no se considera necesario..

El elemento article se usa para marcar contenido independiente, auto contenido, que tendría sentido incluso fuera del contexto de la página actual y que podría sindicarse o distribuirse de forma independiente, por ejemplo: una noticia, un artículo en un blog, un comentario en un foro.
