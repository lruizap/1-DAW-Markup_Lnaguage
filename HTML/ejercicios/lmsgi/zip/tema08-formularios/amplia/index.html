HTML y formularios

Ejemplos de uso de formularios y PHP, de nuevos inputs HTML5.

Es necesario que los ficheros se ejecuten en u nservidor con PHP activo.

Pruebas

  • Probar los ejemplos de clase
  • Probar ampliación HTML5
  • Probar a ejecutar los ejemplos PHP, en local abriendo el fichero directamente en el navegador, y comprobar que no funciona la parte PHP.
  • Ejecutar ejemplos PHP contra servidor (php -S localhost:8080)
  • Modificar extesión .php por .html y ver que no se ejecuta el código PHP
  • Modificar action para probar el uso de get o post. Ver capturas en developer tools del navegador o capturar con Wireshark

Enlaces a ficheros

  • + HTML5: form-html5.html

Ejemplos de uso PHP

  • procesa.php?param1=uno&param2=dos&param3=Hola+Mundo
  • dni.html y dni.php
